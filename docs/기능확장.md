# 보고서 작성 시스템

## 확장 개선

1. Amplify에 정적 파일 형태(React, Vite)로 배포
2. DB랑 연결 시켜서, PDF 저장 시 -> PDF 파일 자체는 다운 / 내용만(FormData) DB에 저장
3. 근데 DB랑 소통을 React서 바로 할 수 있나? Next.js 로 전환해야 하나?
4. 그렇다치고, 그럼 aws RDS랑 연결 -> 저장은 하나의 pdf문서 리스트라는 기준으로 거기에 formData를 저장
5. 사용자는 리스트에서 하나를 선택하면, DB정보를 불러와서 문서 내용 채워넣기를 할 수 있다.
6. 기존 문서를 불러왔을 때 '저장'을 누르면 수정 / new생성 으로 갈건지는 정해야한다.

** 일단 프로토타입이므로, 너무 복잡하게 구성하지 않음 **
